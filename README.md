# FinLedger 
FinLedger is a personal finance dashboard built using React. It helps users track their income, expenses, and overall financial health in a simple and visually interactive way.

##  Features

* Add, edit, and delete transactions
* View total balance, income, and expenses
* Savings rate calculation
* Filter transactions by category, type, and date
* Search functionality for transactions
* Interactive charts for financial insights
* Role-based access (Admin / Viewer)
* Data persistence using localStorage

##  Tech Stack

* React (Vite)
* Context API (for state management)
* Tailwind CSS (for UI styling)
* Recharts (for data visualization)
* React Hot Toast (for notifications)

##  How to Run the Project

1. Clone the repository
   git clone https://github.com/Saloninaik12/finledger.git

2. Go to the project folder
   cd finledger

3. Install dependencies
   npm install

4. Run the app
   npm run dev

## Folder Structure (Important Parts)

* components/ → UI components like Navbar, Sidebar, Charts
* pages/ → Dashboard, Transactions, Insights
* context/ → Global state management
* data/ → Initial transaction data
